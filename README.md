# Tottenham-Hostpur-Injury-Predictor-2
This project builds an ensemble injury risk model for Tottenham Hotspur players using match-by-match performance data scraped from FBref. For each fixture, features such as rolling minutes played, workload spikes, days of rest, consecutive starts, and prior injury history are engineered to produce a pre-match injury risk score. A Logistic Regression and Random Forest classifier are trained and combined into an ensemble, outputting a percentage risk for each player appearance. The model was validated against 26 confirmed new injuries across the 2023–25 seasons, achieving an AUC-ROC of 0.618, and correctly flagged high-risk scores (70–90%+) for players including Dominic Solanke, Wilson Odobert, and Cristian Romero at the time of their actual injuries.
